# FRONT END USING REACT JS

## REQUIREMENT

1. NodeJS version 16.x.x

2. Npm version 8.x.x

## USING

- Step 1: Clone this branch

- Step 2: `cd frontend`

- Step 3: `code .`

- Step 4: In *src/constants/api_config.js*, change `DOMAIN` to your *back-end url*

- Step 5: `npm install`

- Step 6: `npm start`
